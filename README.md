# AI Patient Chatbot

An intelligent conversational agent designed to simulate patient-healthcare provider interactions for educational and training purposes. This chatbot leverages modern AI technologies to provide realistic dialogue scenarios, helping medical students and healthcare professionals practice communication skills in a safe, controlled environment.

## Tech Stack

- **Backend**: Python, FastAPI
- **Frontend**: React, Vite
- **Database**: SQLite
- **AI/LLM**: Local language models (GGUF format)

## Disclaimer

> **⚠️ This project uses synthetic data only and is not intended for medical use.**
